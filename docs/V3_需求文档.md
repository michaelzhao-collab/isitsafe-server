#### A.4 商业模式

### B. 本地化情报推送 + 情报案例合并 Tab

#### B.1 市场痛点
- App 留存差，用户没事不打开
- 国家反诈虽推送但不按地理/人群精准
- 不同人群关心的骗局类型完全不同

#### B.2 实际可做的方案

**内容来源**：用户上报 + 公开数据抓取（国家反诈微博/FTC 公开页）+ AI 改写 + 编辑团队审核

**推送规则**：地理位置 + 人群标签 + 频次（默认 1 条/天）

**首页通知条**：
- HomeEmptyStateContent 上方插入 1 条 banner
- **仅有新通知时显示**，无新通知时不渲染
- 显示「📢 官方提醒：XXX」
- 点击 → 跳到「情报案例」Tab

**情报案例 Tab**（V2 知识库 Tab 改造）：
- 顶部 Segment 切换：**今日情报** / **案例库**
- 「案例库」segment = V2 现有 `KnowledgeView`，完全不动
- 「今日情报」segment = 新增，含三类消息混合展示：
  - 系统全网官方消息（来自管理员编辑团队）
  - 家庭官方消息（来自 E 模块的官方匿名广播）

#### B.3 不做（明确删除）
- ❌ 抓付费/私域内容
- ❌ 以"新闻服务"定位（保持反诈科普）
- ❌ 挑战国家反诈中心权威

#### B.4 关键前提
- **¥80k 内容运营预算（3 个月）必须前置批准**

### E. 家庭守护账号（共享 + 关怀 + 官方消息，**全免费**）

#### E.1 关键定位（v3 重写）

> ❌ **不是**「监控父母手机」
> ❌ **不是**「家庭付费包」
> ✅ **是**「免费家庭关系网络 + 关怀提醒 + 官方匿名广播」

#### E.2 三大核心机制

##### 机制 ①：免费家庭组（关系网络）
- **创建/加入完全免费**，任何用户都可
- 一组最多 5 人
- 邀请方式：微信 / 短信 / Universal Link
- 加入后**不带任何 Pro 权益**（订阅仍按 V2 现有 5 次/天逻辑）

##### 机制 ②：关怀机制（100% 免费）
- App 自动统计「今日是否活跃」
- 活跃 = 用户**真实主动打开 App**任意页面 1 次（Push 唤起不算）
- 家庭页可见每人状态（🟢 已活跃 / ⚠️ 未活跃 N 天）
- 连续 2 天未活跃 → 自动给其他成员发 push："X 已 2 天未活跃，去看看？"
- 连续 3 天未活跃 → push + 短信（**短信免费，限每家庭 1 条/天**）
- 用户重新打开 → 状态恢复 + 取消未发提醒

##### 机制 ③：官方匿名广播
- 家庭任何人在 StarLens 内查询/分享一条信息 → AI 检测 → 以「StarLens 官方」名义按真实结果发给家庭所有人
- **不显示触发者**（保护隐私 + 增加权威感）
- 同一条信息家庭内**一天内只发 1 次**（避免轰炸）
- 三种结果形态：
  - **诈骗**（红）：「已识别诈骗，请家人警惕」
  - **安全**（绿）：「经核实，暂未发现风险」
  - **未知**（黄）：「暂无法确认，请谨慎对待」
- 免费配额：**1 条官方提醒/家庭/天**（不分形态）；超额需付费会员

#### E.3 不能做的（明确删除）
- ❌ **监听父母微信/QQ/抖音等第三方 App 内容**：iOS 沙箱
- ❌ **实时监控通话内容**：iOS 不允许
- ❌ **私人事件流（直接显示 A 查了 XXX）**：改为官方匿名广播
- ❌ **强制家庭付费墙**：家庭组创建 / 加入永久免费
- ⏸ **广播撤回 / 紧急下架**：一期不带，推二期（FamilyBroadcast 无 is_recalled 字段）

#### E.3b 家庭套餐与权益分发（S1-3 / S1-4 已落地）

实际订阅商品：
- `starlens.family.monthly.subscription` ¥28 / $14.99
- `starlens.family.annual.subscription` ¥168 / $129.99
- familyShareable = false（用自建家庭组分发，不走 Apple Family Sharing）

owner 付费 family.* → 全员（owner + members）获 unlimited 配额（查询 + 官方提醒）。
EntitlementService 按订阅 join MembershipPlan.tier 动态判断；owner 退订立即对全员生效。

#### E.4 营销文案约定

❌ **不要这样宣传**：
- "全方位监控父母手机"
- "实时同步家人手机活动"
- "加入家庭组享 Pro 权益"

✅ **应该这样宣传**：
- "免费创建家庭组，全家互相守护"
- "家人若长时间未活跃，自动关怀提醒"
- "StarLens 官方匿名广播，家庭防诈一致信息"

#### E.5 技术依赖
- iOS: Universal Link、APNs（含 background-fetch 静默推送）
- Server: NestJS + Prisma（家庭组/成员/事件/活跃日志/官方消息 五张表）
- **短信网关**：阿里云短信（国内）/ Twilio（海外），用于"3 天未活跃"兜底
- AI 检测：复用 V2 编排（识别结果分类为"诈骗/安全/未知"）
