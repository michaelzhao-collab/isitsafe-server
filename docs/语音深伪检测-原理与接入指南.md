## 🔬 业界主流检测技术（4 类）

### A. 声学特征分析（最经典）

AI 合成语音跟真人语音在**物理声学**上有可量化差异：

| 特征 | 真人 | AI 合成（TTS / Voice Cloning）|
|---|---|---|
| **高频谐波** | 8-20kHz 有丰富细节 | 高频段过度平滑或缺失 |
| **频谱断层** | 自然连续 | 某些频段出现规律性"空洞" |
| **基频抖动 (jitter)** | 声带不稳定，有微小颤抖 | TTS 过于稳定 |
| **共振峰 (formant)** | 因人而异（受声道形状影响）| 模型平均化，特征趋同 |

工具链：音频 → 傅里叶变换（FFT）→ 梅尔频谱图（mel-spectrogram）→ CNN / Transformer 分类

### B. 韵律分析（Prosody）

真人说话的节奏不是匀速：

| 维度 | 真人 | AI |
|---|---|---|
| 停顿位置 | 语义 / 换气驱动，有规律 | 按时长或标点，**机械等距** |
| 重音 | 强调关键词 | 平均分布 |
| 语速变化 | 情绪驱动，有起伏 | 单调匀速 |
| 拼接痕迹 | 无 | 拼接型合成在词与词之间有微小不连续 |

### C. 情感连贯性

AI 表达"开心→悲伤"等情绪转折时**音调跨度突然**，缺乏过渡。真人的情绪曲线是平滑的。

### D. 反检测特征（最前沿）

检测 GAN（生成对抗网络）生成过程中留下的**纹理噪声指纹**——每个 AI 模型在生成时都会留下独特"水印"。

## 📦 StarLensAI 现状（重要）

### 架构：3 层 fallback

`Server/src/modules/deepfake/deepfake.service.ts`:

```
优先 1: Reality Defender（海外）  → 需配 REALITY_DEFENDER_API_KEY
优先 2: 火山引擎（国内）           → 需配 VOLCENGINE_AK/SK + 实现签名
Fallback: stub_v1（伪检测）        → 当前实际跑的就是这条
```

### 真相：**目前还没接真实 AI**

`stubAnalyze()` 是用 URL 末尾字符的 char code 当确定性 hash，按公式返回 `score / label / features`：

```typescript
const seed = (last_char_code + durationSec) % 100;
if (seed < 30) score = 0.05~0.20    // low (安全)
else if (seed < 70) score = 0.30~0.58 // medium (可疑)
else score = 0.75~0.95               // high (高危)
```

返回的 `features` 是写死的 5 条文字："呼吸节奏不自然 / 音色细节缺失 / 情感波动不连续 / 背景噪音模式异常 / 频谱断层" —— **不是真分析的，是按 label 取前 1-3 条展示**。

### 也就是说

| 场景 | 现实 |
|---|---|
| 用户上传 1 个真实录音 | 服务端按 URL 字符算 score，**结果跟音频内容无关**，仅与文件名有关 |
| 用户上传 1 个 AI 克隆语音 | 同上 |
| 同一段录音上传 10 次 | 文件名不同 → 10 个不同结果 |

⚠️ **MVP 阶段可以这样上线先试水**，但要明确告诉用户"AI 仅辅助判断，请通过视频通话或当面核实"（已在 heroCard 文案中标注）。

## 🚀 接入真实检测：2 个推荐 Provider

### 方案 A：Reality Defender（海外用户最优）

#### 步骤

1. 去 https://www.realitydefender.com → 申请账号
2. 选 API plan（约 $0.05/次，按需付费）
3. 后台拿 `API Key`
4. Railway 加 env：
   ```bash
   REALITY_DEFENDER_API_KEY=rd_xxxxx
   REALITY_DEFENDER_API_URL=https://api.realitydefender.com/api/v2/audio/analyze  # 可选
   ```
5. 重启 server（Railway 自动）
6. 用户下次上传音频时，服务端自动走 Reality Defender，而不是 stub

#### 代码层

`deepfake.service.ts:161-188` 的 `callRealityDefender()` 已经写好，**不需要改**。

### 方案 B：火山引擎（国内用户最优）

#### 步骤

1. 注册火山引擎账号 → 实名认证
2. 控制台 → "语音技术" → 开通 "AI 合成语音检测"

5. Railway 加 env：
   ```bash
   VOLCENGINE_AK=xxxx
   VOLCENGINE_SK=xxxx
   ```

#### 代码层：补完火山签名

`callVolcEngine()` 当前是 throw。补完示例（使用 SDK）：

```typescript
// npm i @volcengine/openapi
import { VolcSign } from '@volcengine/openapi';

private async callVolcEngine(fileUrl: string): Promise<{...}> {
  const ak = process.env.VOLCENGINE_AK!;
  const sk = process.env.VOLCENGINE_SK!;
  const endpoint = 'https://open.volcengineapi.com';
  const action = 'AudioDeepfakeDetect';
  const version = '2024-12-01';

  const body = JSON.stringify({ audio_url: fileUrl });
  const headers = VolcSign.sign({
    method: 'POST',
    host: 'open.volcengineapi.com',
    path: `/?Action=${action}&Version=${version}`,
    body,
    accessKey: ak,
    secretKey: sk,
    service: 'speech',
    region: 'cn-beijing',
  });
  const resp = await axios.post(
    `${endpoint}/?Action=${action}&Version=${version}`,
    body,
    { headers, timeout: 30000 },
  );
  const score = Number(resp.data?.Result?.SyntheticScore ?? 0);
  const label: 'low' | 'medium' | 'high' =
    score < 0.30 ? 'low' : score < 0.70 ? 'medium' : 'high';
  return {
    result: {
      score,
      label,
      features: [],  // 火山一期可能不返 features，自己拼
    },
    raw: resp.data,
  };
}
```

## 📊 接入对比 + 推荐

| 维度 | Reality Defender | 火山引擎 |
|---|---|---|
| 海外用户 | ✅ 速度快 | ❌ 延迟高 |
| 国内用户 | ⚠️ 防火墙问题 | ✅ |
| 准确率 | 92-95% | 90%+ |
| 价格 | $0.05/次 | ¥0.1/次（≈ $0.014）|
| 接入难度 | API Key 即可 | 需实现 HMAC 签名 |
| 中文识别 | 较弱 | ✅ 优秀 |

### 短期推荐路线

**Phase 1（先上线试水，1 周内）**：
- 先**接 Reality Defender**（最快，1 天搞定）
- 验证检测效果 + 用户体验
- 看用户实际使用频率决定是否值得保留

**Phase 2（流量稳定后，1 个月内）**：
- 加火山引擎做 fallback（覆盖国内用户）
- 双 provider 并存：Reality Defender 失败 → fallback 火山

**Phase 3（流量大于 10K MAU 时）**：
- 考虑自己用 PyTorch 训一个小模型（开源数据集 ASVspoof 2019/2021 可用）
- 部署成本：自建 GPU 推理服务约 $50/月
- 优势：无 API 限流 + 数据闭环

## ⚠️ 安全与隐私

### 1. CDN URL 验证（已实现）

`assertOwnCdnUrl()` 防止用户传任意 URL → 服务端把它转给第三方 AI → 用户私有数据外泄。

`CDN_DOMAIN` env 必须配 + iOS 上传必须走 R2/CDN。

### 2. 数据保留

- Reality Defender / 火山引擎都会**临时存储**用户上传的音频用于分析（最多 30 天）
- 用户 Privacy Policy 必须提到"音频可能被第三方 AI 服务处理"

### 3. 误报处理

- 检测结果有误判（无论高低）→ 用户可能依赖错误结论
- App 文案必须明确："AI 仅辅助判断，请通过视频通话或当面核实"
- 检测结果页底部有这句已固定文案，不要删
