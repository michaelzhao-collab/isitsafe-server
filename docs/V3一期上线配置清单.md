# V3 一期上线配置清单

> 代码已完成，本文档列出所有需要你（产品/运维）配置的内容。
> 顺序：免费 → 付费 → 需账号开通的第三方。可以分批配置，不影响线上现有功能。

---

## 0. 总体说明

**全部 V3 接口在 `/api/v3/*` 命名空间下，V1/V2 接口完全未动。** 即使第三方密钥一个都不配，V3 也能跑（走 stub 数据），只是没有真实数据/推送/AI 能力。

| 模块 | 不配的影响 |
|---|---|
| 推送 APNs | 家庭关怀提醒走 SMS / 应用内通知，不会推送 |
| SMS 网关 | 家庭关怀逾期、SOS 短信无法发送（应用内逻辑仍正常）|
| HIBP | 暗网泄露走 stub 假数据（演示可用，但不是真实泄露）|
| AES 加密密钥 | 暗网监控只能存哈希，无法解密回显用户邮箱 |
| Reality Defender | 语音深伪检测走 stub 评分（依然能跑流程）|

## 2. 后端环境变量（按需配，未配的走 stub）

把以下变量加到 Railway → Server 服务的 Variables 面板：

### 2.1 暗网监控加密密钥（建议配，否则 HIBP 无意义）

```env
# 用于 AES-256-GCM 加密用户邮箱（HIBP 查询时再解密）
# 32 字节，hex 或 base64 都可
BREACH_AES_KEY=<64位hex字符串>
```

生成方法（mac 终端）：
```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

⚠️ **生成后请用密码管理器保存**。丢失等于所有用户邮箱密文都解不出来。

### 2.2 HIBP（Have I Been Pwned）API

```env
HIBP_API_KEY=<你的 HIBP 订阅 key>
HIBP_API_URL=https://haveibeenpwned.com/api/v3  # 可选，默认这个
```

开通步骤：
1. 打开 https://haveibeenpwned.com/API/Key
2. 用一个常用邮箱订阅 **Pwned 1**（$4/月，足够 V3 一期）
3. 复制 Key（一次性显示，没复制到要重新订阅）

### 2.3 APNs 推送（家庭关怀/紧急提醒）

```env
APNS_TEAM_ID=<10位 Team ID>
APNS_KEY_ID=<10位 Key ID>
APNS_AUTH_KEY=<.p8 文件的全部内容>
APNS_BUNDLE_ID=com.starlens.IsItSafe
APNS_ENV=production  # 上线后用 production；TestFlight 用 sandbox
```

开通步骤：
1. 进 https://developer.apple.com/account → Certificates, Identifiers & Profiles
2. **Keys** → **+** → 勾选 **Apple Push Notifications service (APNs)** → 命名 "StarLens APNs" → Continue → Register
3. 下载 `.p8` 文件（只能下一次！丢了就要重新生成）
4. `APNS_KEY_ID` = 该页显示的 10 位 Key ID
5. `APNS_TEAM_ID` = 右上角 Membership 里的 Team ID
6. `APNS_AUTH_KEY` 把 `.p8` 文件全部内容贴进来（含 `-----BEGIN PRIVATE KEY-----` 行）

### 2.4 阿里云 SMS（国内用户）

```env
ALIYUN_SMS_ACCESS_KEY_ID=<RAM 用户 AK>
ALIYUN_SMS_ACCESS_KEY_SECRET=<RAM 用户 SK>
ALIYUN_SMS_SIGN_NAME=星眸安全  # 你申请通过的签名
ALIYUN_SMS_TEMPLATE_CODE_FAMILY_CARE=SMS_xxxxxxxx
```

开通步骤：
1. 阿里云 → 短信服务 → 国内消息
2. 申请**签名**（企业资质 1-2 天审核）
3. 申请**模板**：
   - 类型：通知短信
   - 内容：`【星眸安全】您的家人 ${name} 已 ${hours} 小时未活动，请联系确认。`
   - 模板审核通过后拿到 TemplateCode
4. RAM → 创建用户 → 给权限 `AliyunDysmsFullAccess` → 拿 AK/SK
5. 配 IP 白名单（Railway 出口 IP 可在 Railway 控制台查）

### 2.5 Twilio SMS（海外用户）

```env
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=<auth token>
TWILIO_FROM_NUMBER=+1xxxxxxxxxx
```

开通步骤：
1. https://www.twilio.com/console → 充值 $20（前期够用）
2. **Phone Numbers** → Buy a number（美国号 ~$1.15/月）
3. 在 Console 首页拿 Account SID 和 Auth Token

### 2.6 Reality Defender（语音深伪检测）

```env
REALITY_DEFENDER_API_KEY=<API Key>
REALITY_DEFENDER_API_URL=https://api.realitydefender.com  # 可选
```

开通步骤：
1. https://www.realitydefender.ai/ → Contact Sales
2. 通常是 enterprise 询价，约 $0.05-0.10/次
3. 等 SDR 联系 → 签 NDA → 拿到沙箱 Key
4. **暂未拿到也没关系，没配就走 stub 评分**

### 2.7 火山引擎（豆包/Volcengine，备用 AI Provider）

> ⚠️ 一期仅留接入点位，签名实现需 SDK，**默认不启用**。如果你已经有 Volcengine 账号想试，先存到环境变量即可，代码会优先 Reality Defender，再 fallback Volcengine（实际目前会直接 throw 走 stub）。

```env
VOLCENGINE_AK=
VOLCENGINE_SK=
```


## 4. iOS Xcode 配置（必做）

### 4.1 Share Extension（语音深伪检测：从微信/iMessage 分享）

代码已在 `iOS/StarLensShareExtension/` 下，但**Xcode target 需要你在 IDE 手动添加**：

1. 打开 `iOS/IsItSafe/IsItSafe1.xcodeproj`
2. File → New → Target → **Share Extension** → 命名 `StarLensShareExtension`
3. 新生成的 target 默认有 `ShareViewController.swift` / `Info.plist` / `MainInterface.storyboard`：
   - **删除** Xcode 默认生成的 `ShareViewController.swift` 和 `Info.plist`
   - 把仓库里 `iOS/StarLensShareExtension/ShareViewController.swift` 和 `Info.plist` 拖进 target（勾 Copy if needed 取消，作为 reference）
4. 选中 target → **Signing & Capabilities** → **+ Capability** → **App Groups**
   - 勾选或新建 `group.com.starlens.IsItSafe.share`
5. **主 App target** 同样加 App Groups capability，勾同一个 group ID
6. Bundle Identifier 建议：`com.starlens.IsItSafe.ShareExtension`
7. Build & Run 主 app，再在微信/iMessage 长按一条语音 → 分享 → 应该能看到 "StarLens 语音深伪检测"

### 4.2 Push Notifications capability

1. 主 App target → Signing & Capabilities → **+ Capability** → **Push Notifications**
2. **+ Capability** → **Background Modes** → 勾 Remote notifications
3. 用真机/TestFlight 测试 push token 注册（模拟器拿不到 device token）

### 4.3 Microphone permission（A1 录音）

`Info.plist` 检查是否已有：
```xml
<key>NSMicrophoneUsageDescription</key>
<string>StarLens 需要使用麦克风录制可疑语音，用于本地深伪检测</string>
```

### 4.4 Universal Links（可选，能让分享链接直跳 App）

只有想做"短信里点击链接直接打开 App"才需要。一期非必须。

## 7. 不需要你配，但你应该知道的事

- **数据库迁移只加列、不删列、不改类型**：V1/V2 SQL 完全兼容
- **Stub 回退机制**：每个第三方都有 stub，缺密钥不会让接口 500
- **所有 V3 endpoint 在 `/api/v3/*`**：旧接口零影响
- **Cron 任务**：家庭关怀检测在 Server 启动后自动运行（每 30 分钟一次），无需额外配置
- **Admin 入口**：V3 情报管理在左侧菜单 `情报中心 V3` 和 `用户上报审批`，已加 AdminRoleGuard
