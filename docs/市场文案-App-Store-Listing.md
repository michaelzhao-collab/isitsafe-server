# ✅ V2 — 优化版（推荐使用，吸收 V1 合规部分 + 双语 + 用户分层）

## V2 设计原则

1. **App Name 含 #1 关键词**（"Scam Detector"）—— App Store 算法对名字字段加权最高
2. **Subtitle 情感诉求 + 场景词**（不是产品描述）
3. **Description 前 3 行 = 钩子**（用户看不到展开按钮，前 3 行决定下载与否）
4. **Keywords 字段塞满 100 字符**，不与 Name/Subtitle 重复
5. **中英双语独立优化**，中文不是英文翻译，要符合本地搜索习惯
6. **保留 V1 法律合规部分**：Apple EULA URL + Terms/Privacy + IMPORTANT NOTICE
7. **APP 主旨重新定位**：从"AI 风险检测工具"→"你的私人反诈助手 + 全家安全网"
